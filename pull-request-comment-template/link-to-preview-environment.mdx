# Link to Preview Environment

Links to the main preview environment of the job linked to the current pull request.
Renders an image containing a screenshot of the root of the preview environment.

## Editor

__Link to Image:__

Image rendered in comment.
Default is screenshot of root of the preview environment.

<br/>
__Alt Text:__

Alt text of image.

<br/>
__Height:__

Height of the image.

<br/>
__Width:__

Width of the image.

<br/>
__Corner Radius:__

Corner radius of the image.

<br/>
__Link:__

The link the associated with the image.
Default is to root of the preview environment.

<br/>
__Align:__

Alignment of the image, `left`, `center` or `right`.
